Este sistema permite la gestión y consulta de una base de datos biológica almacenada en archivos CSV. El programa utiliza Programación Orientada a Objetos (POO) para representar a los animales y controlar las operaciones del zoológico.

## Requisitos
Python 3.x instalado.
Archivos `zoo.csv` y `clases.csv` en el mismo directorio que los scripts.

## Estructura del Proyecto
`ClasesDef.py`: Contiene las definiciones de las clases `Animal` y `Zoologico`.
`main.py`: Interfaz de usuario por consola y flujo principal.
`zoo.csv`: Base de datos de los animales.
`clases.csv`: Catálogo de clasificaciones (Mamíferos, Aves, etc.).
